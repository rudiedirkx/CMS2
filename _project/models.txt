Velden met ref_ in t begin (bijv ref_office) zijn Node ID's.
Het ID kan je dus aanroepen met $node->ref_office
Het object aanroepen kan dan met $node->office
Het object wordt pas geladen als ->office wordt aangeroepen
$node->office is dan weer een Node (met refs)
Lekker schakelen dus: $node->office->manager->title


public function __get( $k ) {
	if ( !property_exists($this, $k) ) {
		if ( $this->{'ref_'.$k} ) {
			$this->$k = Node::load($this->{'ref_'.$k});
		}
		else {
			$this->$k = null;
		}
	}
	return $this->$k;
}
